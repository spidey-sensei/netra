## Critical Calculation Rules

- Closing Balance    : ONLY use the row explicitly labeled "Closing Balance" at the very
                       bottom of the transaction table. Never use any mid-table balance value.

- Opening Balance    : ONLY use the row explicitly labeled "Opening Balance" at the very
                       top of the transaction table.

- Total Deposits     : Add ALL values from the Deposits column row by row.
                       Do not stop early. Include every single deposit entry.

- Total Withdrawals  : Add ALL values from the Withdrawals column row by row.
                       Do not stop early. Include every single withdrawal entry.

- Net Change         : Closing Balance MINUS Opening Balance.
                       Negative means more was spent than received.
