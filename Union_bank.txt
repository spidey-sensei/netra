## Critical: Union Bank Balance and Order Rules

- Transaction order  : Union Bank statements are in REVERSE chronological order.
                       S.No 1 = most recent transaction (end of period)
                       Highest S.No = oldest transaction (start of period)
                       Always be aware of this when answering date-based questions.

- Closing Balance    : Explicitly labeled "Closing Balance" in a highlighted row at the
                       bottom of the LAST page. Use this value directly.
                       It will appear as "8571.17 Cr" format.

- Opening Balance    : NOT labeled. Must be calculated from the LAST row (highest S.No):
                       Last row is Credit (Cr) → Opening = Last row Balance - Credit amount
                       Last row is Debit (Dr)  → Opening = Last row Balance + Debit amount

- Amount format      : Shown as "1000.00 (Cr)" or "2857.00 (Dr)" with brackets
                       Strip the (Cr)/(Dr) and brackets before any arithmetic.

- Balance format     : Shown as "8571.17 (Cr)" with brackets in transaction rows
                       Shown as "8571.17 Cr" without brackets in Closing Balance row


## Union Bank Transaction Table Structure

Each transaction row has these columns:
- S.No           : Serial number (1 = newest, highest = oldest)
- Date           : Transaction date (DD/MM/YYYY)
- Transaction Id : Unique transaction reference (e.g. "S98720935")
- Remarks        : Description of transaction
- Amount(Rs.)    : Transaction amount with (Cr) or (Dr) suffix
- Balance(Rs.)   : Running balance after transaction with (Cr) or (Dr) suffix

Note: Union Bank does NOT have separate Debit/Credit columns.
Everything is in one Amount column with (Cr) or (Dr) to indicate direction.


## How to Categorize Union Bank Transactions

Use these rules to classify each transaction:
- UPIAB / UPI...CR    → UPI Incoming (Credit) — money received
- UPIAR / UPI...DR    → UPI Outgoing (Debit) — money sent
- UPIAR.../REV        → UPI Reversal (refund credited back)
- NACH                → Auto-Debit (loan EMI, insurance — always a debit)
- EMANCH              → ECS/NACH mandate charge or bounce penalty
- IMPSAB              → IMPS Incoming (Credit)
- IMPSAR              → IMPS Outgoing (Debit)
- 6083326880011313/.. → Internal fund transfer / cash withdrawal via branch
- Int.Pd              → Interest credited on savings account
- ATM Usage Charges   → ATM usage fee (debit)
- Sms Charges         → SMS alert charges (debit)
- General Charges Recovery → Bank maintenance charges (debit)
- PhonePe /YESB/phonepepayout → PhonePe payout received (credit)
- Cholamandal / TPCapfrst / IDFC FIRST → Loan EMI auto-debit
