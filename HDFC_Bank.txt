## Critical: Where HDFC Balances Are Located

- Opening Balance : ONLY in the STATEMENT SUMMARY table at the bottom of the LAST page
                    labeled "Opening Balance". Never use any transaction row balance.

- Closing Balance : ONLY in the STATEMENT SUMMARY table at the bottom of the LAST page
                    labeled "Closing Bal". Never use any transaction row balance.

- Total Debits    : ONLY from STATEMENT SUMMARY labeled "Debits". Do not manually sum rows.

- Total Credits   : ONLY from STATEMENT SUMMARY labeled "Credits". Do not manually sum rows.

- Per-transaction balance: The "Closing Balance" column in each transaction row shows
                           the running balance after that transaction — do NOT use this
                           as the final closing balance.


## HDFC Transaction Table Structure

Each transaction row has these columns:
- Date           : Transaction date (DD/MM/YY)
- Narration      : Description of the transaction
- Chq./Ref.No.   : Cheque or reference number
- Value Dt       : Value date
- Withdrawal Amt.: Amount debited (outgoing)
- Deposit Amt.   : Amount credited (incoming)
- Closing Balance: Running balance after this transaction

Note: HDFC does NOT have a separate "Opening Balance" row in the transaction table.
The opening balance is only in the Statement Summary on the last page.


## How to Categorize HDFC Transactions

Use these rules to classify each transaction:
- UPI            → UPI Transfer (person or merchant)
- NEFT DR        → Outgoing Bank Transfer
- NEFT CR        → Incoming Bank Transfer
- IMPS           → Instant Payment (person to person)
- EMI / CHQ      → Loan EMI via Cheque
- ACH D          → Auto-Debit (home loan, insurance EMI)
- IB BILLPAY DR  → Internet Banking Bill Payment
- NHDF / PHDF    → HDFC Bill Payment (credit card, utility)
- NWD / ATW / EAW→ ATM Withdrawal
- POS            → Point of Sale / Debit Card Purchase
- CASH DEP / MICRO ATM CASH DEP → Cash Deposit
- FT - CR        → Fund Transfer Credit (incoming)
- CREDIT INTEREST CAPITALISED → Interest Earned
- CBDT TAX       → Income Tax Payment
- REV-IMPS       → Reversal of IMPS transaction
